# SOC Detection Engineering Lab

## About

This repository contains hands-on Security Operations Center (SOC) detection engineering labs built using **Windows**, **Sysmon**, **Wazuh**, and **MITRE ATT&CK**.

Each lab simulates a real-world attacker technique, captures endpoint telemetry with Sysmon, and demonstrates how defenders can investigate and detect the activity.

The objective of this project is to strengthen practical blue-team skills while building a portfolio of detection engineering and incident investigation exercises.

---

## Lab Environment

### Attacker

* Kali Linux
* Python HTTP Server

### Victim

* Windows 10
* Sysmon
* Wazuh Agent

### SIEM / Detection Platform

* Wazuh
* Sysmon
* MITRE ATT&CK Framework
